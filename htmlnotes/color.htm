Note - There are 3 ways to do Color.
- Name
- Hex
- Styles using RGBA

Name is limited to 140 defined names that you can lookup.

If you use Hex you have over 16 million options.
-Syntax you are going to use a # in front of the color.

Note: Web safe colors use repeated values.

RGBA which stands for: Red, Green, Blue, Alpha.
- Can only be used with stylesheets.
- Syntax will be: rgba(0-255, 0-255, 0-255, 0-1)
